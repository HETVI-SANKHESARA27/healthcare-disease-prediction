# Healthcare AI - Smart Disease Prediction System

An AI-powered healthcare application that predicts multiple diseases using Machine Learning models.

## Features
- Diabetes Disease Prediction
- Heart Disease Prediction
- Parkinson's Disease Prediction
- User Login System
- Interactive Streamlit Interface

## Technologies Used
- Python
- Streamlit
- NumPy
- Machine Learning

## Live Demo
https://healthcare-disease-prediction-2706.streamlit.app

## Project Goal
To help users get quick disease risk predictions based on health parameters using machine learning models.
